Aplikacja webowa inspirowana Pokemon GO o odnajdowaniu krasnali we Wrocławiu. Gra polega na zbieraniu informacji o krasnalach poprzez znajdowanie ich w przestrzeni miejskiej.

### Funkcjonalność
- wykrywanie lokalizacji i możliwość interacji tylko z pobliskimi krasnalami
- tworzenie konta
- zapisywanie postępów
- osiągnięcia

### Stos technologiczny
- baza danych: postgres
- backend: flask
- frontend: React
